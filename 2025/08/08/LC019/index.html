LC019 - 删除链表的倒数第N个节点

原题

19. 删除链表的倒数第 N 个结点 - 力扣（LeetCode）

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

提示：

  • 链表中结点的数目为 sz
  • 1 <= sz <= 30
  • 0 <= Node.val <= 100
  • 1 <= n <= sz

进阶：你能尝试使用一趟扫描实现吗？

题解

双指针

时间复杂度：O(n)，其中 n 是链表的长度。

空间复杂度：O(1)。

官方解法（遍历两次链表获得位置）

时间复杂度：O(n)，其中 n 是链表的长度。

空间复杂度：O(1)。

官方解法（栈）

时间复杂度：O(n)，其中 n 是链表的长度。

空间复杂度：O(n)，其中 n 是链表的长度。主要为栈的开销。

总结

强烈建议加哑节点的情况

这些场景下加 dummy 基本上可以让你少踩一半坑。

  1. 可能删除头节点
    • 删除第 n 个节点（可能 n 等于链表长度）
    • 删除倒数第 n 个节点
    • 条件删除（例如删除所有值等于 x 的节点）
  2. 可能在头部插入新节点
    • 在第 k 个位置插入
    • 排序或重建链表时可能新节点会在最前面
  3. 链表分割或重组
    • 按值拆成两部分再合并
    • 按奇偶位置拆分后拼回
  4. 局部反转可能包含头节点
    • “反转第 m 到第 n 个节点”，且 m 可能是 1
    • 反转整个链表
